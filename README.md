# Calculadora Tabajara 2025

## Introdução

Uma calculadora de console simples que permite realizar as 4 operações básicas, além de visualizar  a tabuada e o histórico de operações.

## Funcionalidades

- **Operações Básicas**: Realize somas, subtrações, multiplicações e divisões.
- **Suporte a Decimais**: Trabalhe com números com casas decimais.
- **Validação de Entrada**: A calculadora garante que apenas operações válidas sejam realizadas.
- **Tratamento de Divisão por Zero**: A calculadora é capaz de validar erros de divisão por zero.
- **Tabuada**: A calculadora é capaz de gerar a tabuada de um número informado.
- **Histórico de Operações**: A calculadora é capaz de armazenar um histórico de operações.
